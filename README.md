## Overview

- An unified interface for 32-bit arithmetic with SSE2, AVX, AVX2 and NEON instructions.
- Builds with GCC, Clang and Visual Studio on x86, AMD64, ARM, AArch64.
- Integrates easily due to its header-only nature.
